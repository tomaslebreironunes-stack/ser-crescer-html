🌱 O que usar no Infantário Ser Crescer na Aldeia?

✅ SVG + CSS — A escolha certa

Mascote do infantário em SVG. Escala perfeitamente em mobile e tablet. Claude Code gera o SVG inteiro em 1 prompt. Leve, sem dependências.

✅ CSS para decoração

Flores, borboletas, nuvens, estrelas em redor do site — CSS shapes animadas. Zero peso extra, Claude faz em segundos.

⚠️ Canvas só se precisares

Se o cliente quiser um mini-jogo ("clica no cogumelo e ganhas pontos") ou chuva de cogumelos no hero. Não para decoração estática.

❌ Three.js — não aqui

Overkill absoluto para um infantário. +600KB a carregar para pais em 4G por uma mascote. Guarda para projectos tech premium.
